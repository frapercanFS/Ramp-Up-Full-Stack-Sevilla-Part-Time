¿Qué hacemos?

Somos alumnos y un profesor que estamos aprendiendo las bases de desarrollo web con HTML, CSS y JavaScript, y proximamente nos adentraremos en el perfil MERN

¿Qué es el perfil MERN?

MERN son las siglas de MongoDB, Express, React, Node, por las cuatro tecnologías clave que componen el perfil.

  • MongoDB - base de datos de documentos
  • Express(.js) - framework web Node.js
  • React(.js) - un framework JavaScript del lado del cliente
  • Node(.js) - el principal servidor web de JavaScript
